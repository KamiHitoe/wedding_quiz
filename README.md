# wedding_quiz


## 利用技術
- frontend
    - Vue.js
    - Bootstrap or tailwindcss
- backend/infra
    - Firestore
    - Firebase


## 機能要件
- frontend
    - 11ページ分の静的ページの表示
    - ユーザから回答のPOSTリクエスト
    - Bootstrapでデコる

- backend
    - firestoreへのAPIによるget/post
    - NoSQLの集計、ランキング化

- infra
    - firebaseによる静的ページのホスティング
